Conversational RAG over COVID-19 Doctor Interviews — Pipeline

Current config: MiniLM embeddings · hybrid (dense + BM25 → RRF) · MMR λ=0.8 · extractive generation · math judge

flowchart TD
  subgraph INGEST["① Ingestion & Processing"]
    A[PDF / DOCX transcripts
20 interviews] --> B[Clean & normalize text]
    B --> C["Q&A-aware chunking
split on 'Interviewer:'
question + answer kept together"]
    C --> D["Extract metadata + topics
doctor · country · specialty · topic"]
  end
  subgraph INDEX["② Hybrid Index (persisted to disk)"]
    D --> E["Dense embeddings
MiniLM · 384-dim"]
    D --> F["BM25 lexical index"]
    E --> G[(Hybrid Store
dense.npy + chunks.json + BM25)]
    F --> G
  end
  Q([User question]) --> R
  subgraph RETRIEVE["③ Hybrid Retrieval Engine"]
    R["Score every chunk:
dense cosine + BM25"] --> S["Reciprocal Rank Fusion (RRF)"]
    S --> T{"Metadata filter?
(country / topic)"}
    T -->|yes| U[Restrict to matching chunks]
    T -->|no| V[Keep all candidates]
    U --> W
    V --> W["MMR re-rank
drop near-duplicates · λ=0.8"]
    W --> X[Top-k chunks + scores]
  end
  G -.loads.-> R
  X --> GATE{"Fallback gate:
best similarity ≥ threshold?"}
  GATE -->|no| FB["⚠️ Fallback answer + reason
(anti-hallucination)"]
  GATE -->|yes| GEN
  subgraph RESPOND["④ Response Engine"]
    GEN["Generate answer
extractive (default) · or LLM"] --> CITE["Attach citations
quoted lines + doctor/country/file"]
  end
  CITE --> ANS([Grounded answer + citations])
  FB --> ANS
  subgraph EVAL["⑤ Evaluation Pipeline (self-scoring)"]
    GD[(Golden dataset
18 Q&A + expected chunks/citations)] --> EV["Run engine on each question"]
    EV --> JUDGE["Judge: math (default) · or LLM-as-judge"]
    JUDGE --> M1["RAG Triad
Context Relevance · Groundedness · Answer Relevance"]
    JUDGE --> M2["Retrieval
Hit Rate · MRR · Recall · Precision"]
    JUDGE --> M3["Grounding
Citation accuracy"]
    JUDGE --> M4["Ops
Fallback correctness · Latency"]
  end
  ANS -.evaluated by.-> EV
